# 🚀 Como rodar o projeto - Conversor-de-Cambio-Inteligente

Siga os passos abaixo para executar o projeto localmente:

**1. Instalar as dependências**  
`npm install`  
**2. Iniciar o servidor de desenvolvimento**  
`npm run dev`  
**3. Acessar no navegador**  

Após iniciar, o terminal exibirá um link (`http://localhost:1234`).

Segure **Ctrl** e clique no link **(Ctrl + Click)** para abrir no navegador  
ou  
Copie e cole o link manualmente na barra de endereço

> Você também pode acessar por: [Conversor de Cambio Inteligente](https://rikelry.github.io/Conversor-de-Cambio-Inteligente/)
